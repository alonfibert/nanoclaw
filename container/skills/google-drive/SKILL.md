# Google Drive Skill

This skill lets you access Google Drive directly. You can list files, search for documents, and get file metadata.

**Usage:**
```bash
node drive.js list          # List recent files
node drive.js search query  # Search for files
node drive.js info fileId   # Get file details
```

Credentials are OAuth-authenticated and stored securely.
