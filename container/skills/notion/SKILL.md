# Notion Skill

Access your Notion workspace. Read and write pages, search databases, and manage content.

Usage:
```bash
node notion.js list-databases          # List all accessible databases
node notion.js search <query>          # Search pages and databases
node notion.js read-page <page-id>     # Read page content
```

The integration needs to be shared with your Notion databases to access them.
